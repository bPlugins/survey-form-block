Create custom survey forms easily with the Survey Form Block plugin.

== Description ==
Effortlessly craft bespoke surveys in Gutenberg. Engage your audience and glean valuable insights. Tailor surveys with versatile customization options.


= Features =
- **Field Type Selection**: Choose from a variety of field types.


= How to use =
- First, install the Survey Form plugin
- Add the Survey Form block from the block category called "Widgets" in the Gutenberg editor.
- You can change block settings from the right-side settings sidebar.
- Enjoy!

* For installation help click on Installation Tab

== Installation ==

= From Gutenberg Editor: =
1. Go to the WordPress Block/Gutenberg Editor
2. Search For **Survey Form Block**
3. Click on the **Survey Form Block** to add the block

= Download & Upload: =
1. Download the **Survey Form Block** plugin (*.zip file*)
2. In your admin area, go to the Plugins menu and click on **Add New**
3. Click on **Upload Plugin** and choose the **`survey-form-block.zip`** file and click on **Install Now**
4. Activate the plugin and Enjoy!

= Manually: =
1. Download and upload the **Survey Form Block** plugin to the **`/wp-content/plugins/`** directory
2. Activate the plugin through the Plugins menu in WordPress
